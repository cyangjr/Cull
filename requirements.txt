# GPU acceleration (optional, but recommended)
# Required for MPS/CUDA batch mode. Without it the pipeline runs on CPU only.
# Install the build that matches your hardware — do NOT use plain `pip install torch`
# on NVIDIA as it will install the CPU-only build.
#
#   Apple Silicon (MPS):
#     pip install torch torchvision
#
#   NVIDIA CUDA 12.x:
#     pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
#
#   NVIDIA CUDA 11.8:
#     pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
#
#   CPU-only (no GPU):
#     pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
#
# Minimum: torch>=2.1 (required for torch.fft.fft2 and stable MPS support)
